# CDPSession.detach() method

Detaches the cdpSession from the target. Once detached, the cdpSession object won't emit any events and can't be used to send messages.

#### Signature:

```typescript
class CDPSession {
  detach(): Promise<void>;
}
```

**Returns:**

Promise&lt;void&gt;
