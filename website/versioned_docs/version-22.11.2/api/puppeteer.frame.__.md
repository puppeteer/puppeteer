# Frame.$$() method

Queries the frame for all elements matching the given selector.

#### Signature:

```typescript
class Frame {
  $$<Selector extends string>(
    selector: Selector,
    options?: QueryOptions
  ): Promise<Array<ElementHandle<NodeFor<Selector>>>>;
}
```
