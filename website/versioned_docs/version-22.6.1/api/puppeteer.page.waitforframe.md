# Page.waitForFrame() method

Waits for a frame matching the given conditions to appear.

#### Signature:

```typescript
class Page {
  waitForFrame(
    urlOrPredicate: string | ((frame: Frame) => Awaitable<boolean>),
    options?: WaitTimeoutOptions
  ): Promise<Frame>;
}
```
