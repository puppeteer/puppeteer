# EventEmitter.once() method

Like `on` but the listener will only be fired once and then it will be removed.

#### Signature:

```typescript
class EventEmitter {
  once<Key extends keyof EventsWithWildcard<Events>>(
    type: Key,
    handler: Handler<EventsWithWildcard<Events>[Key]>
  ): this;
}
```
